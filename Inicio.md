# REACT + TYPESCRIPT

## Instalar node


## Crear proyecto
npm create vite

## Ejecutar proyecto
npm run dev

## Estructura del proyecto

```
my-react-app/
├── node_modules/
├── public/
│   ├── favicon.ico
│   ├── index.html
│   ├── logo192.png
│   ├── logo512.png
│   ├── manifest.json
│   └── robots.txt
├── src/
│   ├── componentes/
│   │   ├── Header.tsx
│   │   ├── Footer.tsx
│   │   └── MainContent.tsx
│   ├── App.css
│   ├── App.test.tsx
│   ├── App.tsx
│   ├── index.css
│   ├── index.tsx
│   ├── logo.svg
│   ├── react-app-env.d.ts
│   ├── reportWebVitals.ts
│   └── setupTests.ts
├── .gitignore
├── .swcrc
├── package.json
├── tsconfig.json
├── README.md
└── yarn.lock
```

La base está escrita en App.txs.
Cada componente debe escribirse en su propio fichero txs dentro de una carpeta llamada "components" dentro de la carpeta "src".
